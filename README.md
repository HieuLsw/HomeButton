## Installation

### CocoaPods

HomeButton is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'HomeButton'
```

### Manually

Drag the files in `HomeButton/Classes` into your project.
